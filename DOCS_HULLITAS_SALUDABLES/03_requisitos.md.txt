- REQUISITOS FUNCIONALES (RF) — 20

RF01. El sistema permitirá el registro público únicamente de dueños de mascotas.

RF02. El administrador podrá crear cuentas de veterinarios, cuidadores, agendadores y otros administradores autorizados.

RF03. El sistema asignará roles y permisos según el tipo de usuario.

RF04. El sistema permitirá iniciar sesión mediante usuario y contraseña.

RF05. El dueño podrá registrar mascotas con datos básicos como nombre, raza, edad, peso y foto.

RF06. El sistema permitirá consultar la información de las mascotas registradas.

RF07. El dueño podrá solicitar citas veterinarias según disponibilidad del sistema.

RF08. El agendador podrá gestionar la disponibilidad de citas y organizar la agenda de los veterinarios.

RF09. El agendador podrá notificar a los veterinarios sobre sus citas, cambios y disponibilidad horaria.

RF10. El veterinario podrá consultar las citas que tiene asignadas.

RF11. El veterinario podrá registrar diagnósticos, tratamientos, vacunas y observaciones.

RF12. El dueño podrá consultar el historial clínico de sus mascotas.

RF13. El sistema permitirá reservar servicios de guardería para mascotas.

RF14. El administrador podrá gestionar usuarios (excepto dueños), mascotas, citas y servicios.

RF15. El sistema registrará acciones realizadas por administradores y agendadores para control y auditoría.

RF16. El cuidador podrá consultar la lista de mascotas hospedadas en la guardería asignada durante su turno.

RF17. El cuidador podrá registrar actividades de cuidado realizadas a las mascotas, como alimentación, recreación o descanso.

RF18. El cuidador podrá consultar información sobre dietas especiales, alergias y cuidados particulares de las mascotas alojadas.

RF19. El cuidador podrá reportar incidentes o emergencias relacionadas con las mascotas bajo su cuidado.

RF20. El sistema permitirá registrar la entrada y salida de las mascotas en la guardería, almacenando fecha y hora de cada movimiento.

- REQUISITOS NO FUNCIONALES (RNF) — 15

RNF01. El sistema debe garantizar la seguridad de la información almacenada.

RNF02. El acceso a la creación de usuarios administrativos, agendadores y cuidadores estará restringido únicamente a usuarios autorizados.

RNF03. El sistema debe manejar autenticación segura mediante credenciales.

RNF04. La información de usuarios y mascotas debe almacenarse protegiendo la privacidad de los datos.
